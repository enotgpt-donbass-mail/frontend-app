# Система управления очередью: Frontend

**Современное web-приложение для управления очередью и взаимодействия с клиентами.**

### **Описание**

- **Цель проекта:** Обеспечение удобного и эффективного взаимодействия клиентов с системой обслуживания.
- **Ключевые особенности:**
  - Панель администратора для управления системой.
  - Веб-интерфейс для пользователей, включая бронирование и получение информации.
  - Интеграция с Telegram-ботом для удобного взаимодействия.
  - Визуализация данных и аналитика.

### **Установка и запуск**

1. **Клонирование репозитория:** `git clone `
2. **Установка зависимостей:** `npm install`
3. **Запуск в режиме разработки:**`npm run dev`

### **Структура проекта**

- **app:** Routing страниц и основные компоненты страниц
- **components:** Компоненты пользовательского интерфейса
- **features:** Модули с конкретной функциональностью (например, бронирование, история посещений)
- **api:** Взаимодействие с backend API
- **redux:** Stores, slices, providers менеджера состояний redux

### **Технологии**

- **Стандарты кодирования:** ESLint, Prettier
- **Фреймворки:** Next.js, RTK Query
- **Библиотеки стилей:** Material UI
